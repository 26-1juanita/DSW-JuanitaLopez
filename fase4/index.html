Propósito del Sitio


Este sitio tiene como propósito fomentar y fortalecer el aprendizaje de HTML, enfocados en las Etiquetas Básicas de HTML, atributos y valores


Etiquetas Básicas de HTML


HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML.


Atributos y valores


Los atributos y los valores van enlazados, ya que uno depende del otro, ya que la estructura básica de codificación HTML, las etiquetas que se denominan como tags van enlazadas con los atributos y estas se encuentran en la etiqueta inicial. Adicionalmente se encuentran los valores que son algunos que se encuenteran encerrados en las comillas, ya sean simples o dobles.
